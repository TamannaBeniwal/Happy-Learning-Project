#  Happy Learning – An Interactive Learning Platform  
**Status: Work In Progress (Frontend under development, Backend coming soon)**  

Happy Learning is a modern and user-friendly learning platform designed to provide students with an enjoyable, interactive, and simple way to gain knowledge.
The project focuses on delivering an attractive UI, a smooth user experience, and plans to integrate advanced backend features in the future.


##  Features (Currently Available)
-  Clean & modern user interface  
-  Styling using Tailwind CSS  
-  Responsive design 
-  Fast performance with React  
-  Modular & scalable folder structure  


##  Technologies Used

### Frontend
- React (Create React App)
- JavaScript
- Tailwind CSS
- CSS
- HTML

### Backend (Planned)
- Node.js + Express (to be added)
- MongoDB 
- JWT authentication (planned)
- REST APIs (planned)

## Installation & Setup (How to Run the Project)

### Clone the repository:-
`git clone https://github.com/TamannaBeniwal/Happy-Learning-Project.git`

### Go to the project directory:-
`cd Happy-Learning-Project`

### Install dependencies:-
`npm install`

### Start the development server:-
`npm start`
